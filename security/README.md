# 安全模块

包含功能：

1. 邮箱黑名单管理
1. 验证码
    - google reCAPTCHA
    - Cloudflare 智能验证码
1. 密码策略
    - 最小长度
    - 要求字符类型
    - 密码拒绝
        - 泄漏的密码：拒绝之前在泄漏数据库中发现的密码
        - 限制低安全性短语
            - 重复或连续字符
            - 用户信息
            - 自定义词汇（服务名、公司名等）
1. 锁定设置
    - 是否开启
    - 最大失败尝试次数
    - 锁定时长
    - 手动解锁：通过标识符（邮箱、电话号码、用户名）解锁
