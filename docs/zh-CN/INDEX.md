## 🎯 快速查找

### 我想开始开发
→ 阅读 [快速开始指南](DOCKER-QUICKSTART.md)
→ 运行 `./start-all.sh`

### 我想部署到生产环境
→ 阅读 [生产环境快速参考](README-PRODUCTION.md)
→ 完成 [生产环境检查清单](PRODUCTION-CHECKLIST.md)
→ 运行 `./deploy-prod.sh --with-monitoring`
